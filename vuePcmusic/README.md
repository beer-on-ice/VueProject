# 需配合网易云api使用
-	地址[https://github.com/Binaryify/NeteaseCloudMusicApi](https://github.com/Binaryify/NeteaseCloudMusicApi)

## 安装

`
$ git clone git@github.com:Binaryify/NeteaseCloudMusicApi.git

$ npm install
`
## 运行
$ node app.js 


# 在vue中
## 安装
$ npm install

## 运行
npm run dev

1. 音乐搜索
2. 音乐播放
3. 音乐切换
4. 音乐详情
5. 音乐歌词滚动展示
